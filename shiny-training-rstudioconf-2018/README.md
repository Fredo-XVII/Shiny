# shiny-training-rstudioconf-2018

Slides and demo materials for the 2-day Intermediate Shiny Workshop at rstudio::conf 2018 Training Days

## Required packages

- cranlogs
- DT
- flexdashboard
- gapminder
- jsonlite
- lubridate
- miniUI
- shiny
- shinyBS
- shinydashboard
- shinyjs
- shinythemes
- shinytoastr
- tidyverse (specifically using dplyr, ggplot2, stringr)
